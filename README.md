# Практическое задание №2 - TODO API с парсером

Проект на FastAPI + SQLModel, который:

* хранит и возвращает задачи (tasks);
* парсит товары с сайта letu.ru через Playwright и добавляет задачи по результатам парсинга;
* отправляет обновления клиентам по WebSocket.

## Функции

* CRUD для задач (`/tasks`).
* WebSocket-канал `/ws/tasks` для получения событий (создание/обновление/удаление задач).
* Фоновый генератор задач, который периодически запускает парсер и добавляет новые задачи в БД.
* Ручной запуск парсера через `/task-generator/run`.
